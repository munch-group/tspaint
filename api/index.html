Function reference

Painting local ancestry

The high-level entry point: fit the ancestry CTMC on the reference tips and paint the query haplotypes. paint returns a Painting; fit exposes the EM directly.

paint Infer soft local ancestry along query haplotypes from a tree sequence.
Painting Result of :func:paint: the soft local-ancestry posteriors plus the fitted model.
WindowedPainting Handle over a windowed, streamed painting written to a directory by
fit Blocked EM for (Q, π, {w_i}) (CLAUDE.md §3, §11.1.5-6).
FitResult Fitted parameters returned by :func:fit.

Output & segmentation

The soft per-haplotype posterior (Segment tracks) and its collapse into hard ancestry tracts for downstream tract-length / admixture-dating analysis. SoftTrack is the model-agnostic read-out surface (segments / posterior_at / plot) that both Painting and GhostResult subclass; compare_tracks plots several painters side by side.

output.posterior_table Per-sample ancestry posterior as contiguous segments covering [0, L).
output.hard_segments Collapse a soft posterior track into hard ancestry segments.
output.Segment A contiguous span carrying one soft ancestry posterior.
output.missing_info_mask Per-sample spans where the tree is uninformative (isolated sample).
output.posterior_at Posterior vector for sample at a genomic position.
SoftTrack Mixin providing the shared read-out surface for a per-locus soft posterior along the genome.
track.SegmentTrack Wrap any per-sample segments in a plottable track — hard tuples or soft posteriors.
track.compare_tracks Stack several tools’ calls for one haplotype, one row per tool, for visual comparison.

Reference QC & ghost / archaic introgression

Two user-facing tasks. Task 1 — reference QC: audit a panel for admixture / mislabelling and control it, with reference_qc (a single tree sequence or a SINGER ensemble) whose result yields the actionable .soft_refs() / .mask() for a cleaned re-paint (paint(soft_refs=…) down-weight or paint(mask=…) fragment masking), plus foreign_tracts (anonymous foreign tracts; min_depth= gives the fast deterministic deep-ghost flag). Task 2 — ghost search: detect_ghost, the reference-free depth-emission HMM, for accurate calibrated per-locus P(ghost) — accepts a SINGER ensemble and a calibration-robust depth="rank". Both read the leave-one-out map (tspaint.introgression, tspaint.archaic). (detect_archaic is a deprecated alias of detect_ghost.)

reference_qc Audit a reference panel for admixture / mislabelling (Plan A Workflow 1; CLAUDE.md §9).
introgression.ReferenceQC Result of :func:reference_qc — a per-reference admixture/mislabel audit.
foreign_tracts Anonymous foreign-tract inference: where a haplotype is foreign to its expectation
output.loo_posterior_table Per-sample leave-one-out ancestry posterior as contiguous segments.
introgression.foreignness_track Per-sample foreignness components as contiguous segments covering [0, L).
introgression.ForeignnessSegment A contiguous span carrying the per-locus foreignness components for one sample.
detect_ghost Reference-free ghost / archaic introgression detection via a depth-emission HMM.
GhostResult Result of :func:detect_ghost — per-locus P(ghost) from the depth-emission HMM.

Accuracy, calibration, and fragmentation / tract-length metrics for simulated-truth experiments (the tspaint.metrics namespace).

validate.per_base_accuracy Span-weighted per-base painting accuracy.

validate.switch_density Ancestry switches per unit length.
validate.tract_boundary_error Tract-boundary localization error against the truth.
validate.breakpoint_flicker Posterior discontinuity across a sample’s segment boundaries (CLAUDE.md §7.3).
validate.accuracy_by_segment_size Per-base painting accuracy stratified by true ancestry-tract length.
validate.global_proportion Painter’s estimated global ancestry proportion of state.

Comparators (head-to-head)

Painters scored on equal footing by a uniform harness — the full method, a runnable ARG-native baseline, and the RFMix incumbent (the tspaint.compare namespace).

compare.tspaint_paint Paint queries with the full tspaint method (EM fit + down-pass posterior).
compare.nearest_reference_paint Paint queries by their nearest labelled reference (ARG-native baseline).
compare.rfmix_paint RFMix painter with the standard painter(ts, labels, queries) signature.
compare.head_to_head Score a panel of painters on one admixture scenario across ARG substrates.
compare.score_painter Run a painter on ts and score it against the truth.

VCF benchmark harness

Run the field comparators (RFMix, gnomix, SALAI-Net, Recomb-Mix) from VCF input through one uniform contract and score them against simulated truth (the tspaint.benchmark namespace; CLI tspaint benchmark). run shells out to a tool’s own pixi env and parses its calls into a tspaint painting .npz; export_vcf writes query / reference VCFs + a re-keyed truth table from a simulation; score builds the leaderboard; setup / tool_status provision and report the git-only tools.

benchmark.run Dispatch to a painter by name ("tspaint" or an external comparator).
benchmark.export_vcf Write query/reference VCFs, a sample map, and a truth table from a tree sequence.
benchmark.score Score benchmark paintings against a truth table — the head-to-head leaderboard (CLAUDE.md §9).
benchmark.setup Clone + build the git-only comparators listed in the manifest.
benchmark.tool_status Resolved path + availability for every benchmark tool (incl. RFMix).

Input front ends

Obtain a tspaint-ready tree sequence from genotypes (the tspaint.io namespace). The inference front ends tsinfer, singer and argweaver take the same source (dispatched by source_kind) — a tree sequence, a VCF Zarr, or a VCF; relate converts Relate .anc / .mut output; singer_window / singer_windowed stream SINGER over genome windows. estimate_ne gives the whole-panel Ne SINGER’s coalescent prior needs. (The pre-unification names infer_tree_sequence / singer_tree_sequences / convert_relate remain as deprecated aliases.)

io.singer_windowed Sample posterior ARGs for a long region on ONE machine: window SINGER, then stitch.
io.argweaver Sample posterior ARGs from genotypes via ARGweaver (an alternative to :func:singer).

io.check_persistence Run the §5.1 persistence go/no-go on an already-loaded tree sequence.
io.source_kind Classify a front-end source as "ts", "zarr" or "vcf".
io_genotypes.Variants A tool-agnostic biallelic variant matrix (one column per haplotype).

Horizontal BP/EP smoother

The §7 horizontal coupling — a genome-axis forward-backward over each tip’s per-tree beliefs that suppresses spurious switches on inferred ARGs (the tspaint.bp namespace; also paint(smooth=True)).

bp.bp_paint tspaint painter with horizontal BP smoothing.
bp.bp_smooth Forward-backward over per-segment beliefs along the genome.
bp.bp_smooth_track Smooth one tip’s :class:~tspaint.output.Segment track along the genome.
bp.bp_vs_deadband_experiment Compare BP smoothing vs the deadband for segmentation, over seeds.

Simulation (known truth)

msprime admixture scenarios with census-event ground truth — the primary validation track. Call an admixture_demography_* builder (its populations are role-tagged), pass it to simulate_admixture, and get a Simulation back — .ts (optionally mutated), .queries, .labels, .truth_states and .sample_sets — with no population-name constants needed.

Generators & model

The ancestry CTMC generator, transition matrices, stationary distribution, and the tip emission model — including MaskedEmissions, the position-dependent emission that lets a reference be masked (emit the query emission) over flagged spans, i.e. fragment masking (paint(mask=) / fit(mask=), fed from ReferenceQC.mask() / foreign_tracts).

make_generator_2state Build the 2-state ancestry CTMC generator (CLAUDE.md §2.1).

model.stationary_distribution Stationary distribution π of a CTMC generator.
model.tip_emission Soft-clamp emission for a labelled tip (CLAUDE.md §2.2).
model.query_emission Flat / root-frequency emission for an unlabelled query tip (CLAUDE.md §2.2).
model.MaskedEmissions Position-dependent Felsenstein emissions — fragment masking (CLAUDE.md §2.3).
model.validate_generator Check that Q is a valid CTMC generator.
