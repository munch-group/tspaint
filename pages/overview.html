Overview

tspaint infers soft, calibrated local ancestry along individual haplotypes from an inferred tree sequence (Relate → tskit, or tsinfer/SINGER). Ancestry is modelled as a discrete character evolving up the branches of each marginal tree under a continuous-time Markov chain (a CTMC “mugration” model); reference haplotypes label tips through a soft noise model with learned per-tip credibility; and the parameters are fit by EM whose sufficient statistics are accumulated per tree-sequence edge, span-weighted — so a clade persisting across many trees is counted once. The output is, for every haplotype at every position, a calibrated posterior P(ancestry) — soft, not a hard call.

A separate, optional path — admixture dating — reuses the same engine with a time-inhomogeneous generator to estimate when the labelled ancestries diverged or exchanged genes, as directional cross-ancestry rates q_AB(t), q_BA(t) through time. It is its own deliverable and does not change the painting.

See the model for the generative construction, inference for the EM math, admixture dating for the rate-through-time estimator, and assumptions for the sharp edges.

Quickstart

import tspaint

# 1. Get a tree sequence — simulate admixture with known truth,
#    or build one from genotypes: tspaint.io.infer_tree_sequence / singer_tree_sequences.
ts = tspaint.simulate_admixture(n_admix=10, n_ref=10, sequence_length=2e6,
                              T_admix=100, T_split=5000, Ne=1000, random_seed=1)

# 2. Label the reference haplotypes (sample-node id -> ancestry state); paint the rest.
labels = {...}                              # source-A tips -> 0, source-B tips -> 1
painting = tspaint.paint(ts, labels)          # EM-fit on references, paint the queries

painting.posteriors[q]                      # soft per-position posterior (Segments)
painting.segments(deadband=0.4)             # hard ancestry tracts (tract-length distribution)

# 3. Optional: admixture dating — a separate deliverable, reusing the painting's fit.
rtt = painting.rate_through_time()          # or tspaint.fit_rate_through_time(ts, labels)
rtt.plot()                                  # q_AB(t), q_BA(t): when the ancestries diverged / mixed

On an inferred (tsinfer / Relate) ARG, add smooth=True to suppress tree-inference-induced spurious switches with the horizontal BP smoother.

Two routes lead to admixture timing: the tract-length distribution from painting.segments(...) (an indirect, downstream input to pulse-dating methods), and the direct rate-through-time estimator fit_rate_through_time, which fits the cross-ancestry rate as a function of time.
